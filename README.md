# 闪亮狼狼

## 介绍
一个Minecraft Java版游戏模组。
添加了一些可合成宠物相关物品，让宠物变得更闪亮强大。  

## 版本、模组加载器与兼容性
- Minecraft版本 1.21.8-1.21.11
- 可用于Fabric服务端/客户端

## 物品
- 敏捷骨头 提升驯服狼的速度 骨头+糖
- 黄金骨头 驯服狼恢复一半血量，并持续发光。 骨头+金粒
- 爱之骨头 服用后立即恢复为可繁殖状态 骨头+甜浆果
- 命名骨头 在铁砧上命名后给狼起名 骨头+纸
- 听话骨头 对野生狼使用可立即驯服，恢复狼的全部生命值。 骨头+生羊肉
- 力量骨头 短暂提升驯服狼的攻击和防御（力量+抗性提升） 骨头+熟羊肉
所有骨头的合成模式均为十字形
B是骨头，#是材料。
```
pattern: [
  " # ",
  "#B#",
  " # "
]
```
