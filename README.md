# 闪亮狼狼


## 介绍  
添加了一些可合成多彩骨头，让狼变得更闪亮强大。  
这是一个 Minecraft Java 版本模组，适用于 Fabric 模组加载器。


## 物品
- 遗弃骨头 遗弃驯服的狼变成野生状态
- 敏捷骨头 提升驯服狼的速度
- 黄金骨头 驯服狼恢复一半血量，并持续发光。
- 爱之骨头 服用后立即恢复为可繁殖状态
- 命名骨头 在铁砧上命名后给狼起名  
- 听话骨头 对野生狼使用可立即驯服，恢复狼的全部生命值。
- 力量骨头 短暂提升驯服狼的攻击和防御（力量+抗性提升）


## 配方
所有骨头的合成模式均为“十字形”
```
pattern: [
  " # ",
  "#B#",
  " # "
]
```
其中：
- `B` 代表 `minecraft:bone`
- `#` 代表不同的材料
<br/>
| 骨头类型 | 材料（#） | 合成结果 | 数量 |
|----------|-----------------|----------------|------|
| 遗弃骨头 | 腐肉 | `shiny-wolf:abandon_bone` | 3 |
| 敏捷骨头 | 糖 | `shiny-wolf:agile_bone` | 3 |
| 黄金骨头 | 金粒 | `shiny-wolf:gold_bone` | 3 |
| 爱之骨头 | 甜浆果 | `shiny-wolf:love_bone` | 3 |
| 命名骨头 | 纸 | `shiny-wolf:name_bone` | 3 |
| 听话骨头 | 生羊肉 | `shiny-wolf:obedient_bone` | 3 |
| 力量骨头 | 熟羊肉 | `shiny-wolf:power_bone` | 3 |


## 兼容性
- 代码在服务端执行 兼容多人游戏
- Minecraft版本 1.21.x
- 需安装Fabric模组加载器 和 Fabric API
